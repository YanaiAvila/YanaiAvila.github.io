HoloTouch

For my senior design project, my group developed a hand gesture-controlled holographic interface using MediaPipe for real-time hand tracking and Python APIs for computer control, enabling users to manipulate 3D objects, navigate web browsers,and perform actions like moving their cursor, clicking, scrolling, and typing — entirely hands free.

The 3 core parts of this project were:
(1) Hand tracking
(2) Gesture control
(3) Holographic display
